Học Java

Chào mừng bạn đến với lộ trình học tập Java chính thức. Khám phá các khóa học từ cơ bản đến chuyên sâu.

Chạy ứng dụng Java đầu tiên của bạn

Bắt đầu với Java

Tìm hiểu các bước cơ bản để cài đặt JDK và chạy chương trình đầu tiên của bạn.

Chạy chương trình mã nguồn đơn lẻ

Sử dụng tính năng chạy trực tiếp tệp .java mà không cần biên dịch thủ công.

JShell - Công cụ Shell của Java

Tìm hiểu cách sử dụng REPL để thử nghiệm mã Java một cách nhanh chóng.

Làm quen với ngôn ngữ

Cơ bản về ngôn ngữ Java

Biến, kiểu dữ liệu, toán tử và cấu trúc điều khiển trong Java.

Lớp và Đối tượng

Tìm hiểu về lập trình hướng đối tượng (OOP) thông qua Class và Object.

Records - Mô hình dữ liệu bất biến

Sử dụng tính năng Records để định nghĩa các lớp lưu trữ dữ liệu gọn gàng.

Số và Chuỗi

Cách xử lý dữ liệu kiểu số và làm việc với văn bản trong Java.

Làm chủ API

Collections Framework

Học cách sử dụng List, Set, Map và các cấu trúc dữ liệu quan trọng.

Stream API

Xử lý tập hợp dữ liệu một cách hiệu quả và hiện đại theo phong cách functional.

Java I/O API

Cách đọc và ghi dữ liệu từ tệp tin và các luồng (streams).
