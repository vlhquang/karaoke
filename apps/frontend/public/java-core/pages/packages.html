Packages

Package là gì?

Package là một cơ chế nhóm các lớp và interface liên quan lại với nhau. Chúng tương tự như thư mục trong hệ thống tệp tin. Package giúp tránh xung đột tên lớp và tổ chức code tốt hơn.

Khai báo package

// Dòng đầu tiên của tệp Java
package com.example.myapplication;

public class MyClass {
    // ...
}

Import package

// Import một lớp cụ thể
import java.util.ArrayList;

// Import toàn bộ package
import java.util.*;

// Static import
import static java.lang.Math.PI;
