Sử dụng Pattern Matching

Pattern Matching là gì?

Pattern matching là sự tiến hóa lớn tiếp theo của ngôn ngữ Java. Nó cho phép bạn kiểm tra, hủy cấu trúc (deconstruct) và trích xuất dữ liệu từ đối tượng một cách gọn gàng hơn.

Pattern Matching cho instanceof

// Cách cũ (verbose)
if (obj instanceof String) {
    String s = (String) obj;
    IO.println(s.length());
}

// Pattern Matching (Java 16+)
if (obj instanceof String s) {
    IO.println(s.length()); // s tự động được ép kiểu
}

Pattern Matching cho switch

// Java 21+
Object shape = new Circle(5.0);

String result = switch (shape) {
    case Circle c -> "Hình tròn bán kính " + c.radius();
    case Rectangle r -> "Hình chữ nhật " + r.width() + "x" + r.height();
    case null -> "Không có hình dạng";
    default -> "Hình khác";
};
