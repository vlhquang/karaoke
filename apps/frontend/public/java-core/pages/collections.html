Collections Framework

Collections Framework là gì?

Java Collections Framework (JCF) là một kiến trúc hợp nhất để lưu trữ và xử lý các nhóm đối tượng. Nó cung cấp các giao diện, triển khai và thuật toán.

Các giao diện cốt lõi

// List - Danh sách có thứ tự, cho phép trùng lặp
List<String> list = new ArrayList<>();
list.add("Apple"); list.add("Banana"); list.add("Apple");

// Set - Tập hợp không trùng lặp
Set<String> set = new HashSet<>();
set.add("Apple"); set.add("Apple"); // Chỉ có 1 "Apple"

// Map - Ánh xạ khóa-giá trị
Map<String, Integer> map = new HashMap<>();
map.put("Alice", 30);
map.put("Bob", 25);
IO.println(map.get("Alice")); // 30

Các lớp triển khai thông dụng

  • ArrayList: List dựa trên mảng động, truy cập nhanh.
  • LinkedList: List dựa trên danh sách liên kết, thêm/xóa nhanh hơn.
  • HashSet: Set không có thứ tự, hiệu suất cao nhất.
  • TreeSet: Set được sắp xếp.
  • HashMap: Map không có thứ tự, hiệu suất cao nhất.
  • LinkedHashMap: Map giữ thứ tự thêm vào.

Duyệt qua Collection

List<String> fruits = List.of("Apple", "Banana", "Cherry");

// Dùng for-each
for (String fruit : fruits) {
    IO.println(fruit);
}

// Dùng forEach + Lambda
fruits.forEach(fruit -> IO.println(fruit));

// Dùng Iterator
Iterator<String> it = fruits.iterator();
while (it.hasNext()) {
    IO.println(it.next());
}
