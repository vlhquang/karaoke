Các câu lệnh điều khiển luồng

Các câu lệnh trong mã nguồn của bạn thường được thực thi từ trên xuống dưới, theo đúng thứ tự mà chúng xuất hiện. Tuy nhiên, các câu lệnh điều khiển luồng cho phép bạn phá vỡ sự thực thi tuần tự đó bằng cách sử dụng các cấu trúc rẽ nhánh, lặp và ra quyết định.

Câu lệnh If-Then và If-Then-Else

Câu lệnh if-then là câu lệnh điều khiển luồng cơ bản nhất. Nó chỉ thực thi một khối mã nếu một điều kiện cụ thể là đúng (true).

if (isMoving) {
    currentSpeed++;
}

Câu lệnh if-then-else cung cấp một đường dẫn thực thi phụ khi biểu thức "if" có giá trị là false.

if (testScore >= 90) {
    grade = 'A';
} else if (testScore >= 80) {
    grade = 'B';
} else {
    grade = 'F';
}

Câu lệnh Switch

Câu lệnh switch cho phép thực thi nhiều đường dẫn tiềm năng khác nhau dựa trên giá trị của một biến. Nó hoạt động với các kiểu dữ liệu nguyên thủy (thường là int, char), Enum, và String.

(Chi tiết hơn về Switch sẽ có ở bài học riêng)

Các câu lệnh lặp (While, Do-While và For)

Câu lệnh while thực thi một khối lệnh liên tục khi một điều kiện cụ thể còn đúng:

while (count < 11) {
    IO.println("Count is: " + count);
    count++;
}

Câu lệnh do-while tương tự như while, nhưng đảm bảo rằng khối mã sẽ được thực hiện ít nhất một lần:

do {
    IO.println("Count is: " + count);
    count++;
} while (count < 11);

Câu lệnh for cung cấp một cách gọn gàng để lặp qua một phạm vi giá trị:

for (int i=1; i < 11; i++) {
    IO.println("Count is: " + i);
}

Câu lệnh rẽ nhánh (Break, Continue, Return)

  • break: Thoát khỏi vòng lặp hiện tại hoặc câu lệnh switch ngay lập tức.
  • continue: Bỏ qua phần còn lại của lần lặp hiện tại và bắt đầu lần lặp tiếp theo.
  • return: Thoát khỏi phương thức hiện tại và có thể trả về một giá trị.
