JPackage - Đóng gói ứng dụng

JPackage là gì?

jpackage là công cụ tạo trình cài đặt (installer) native cho ứng dụng Java, tức là file .msi/.exe trên Windows, .dmg/.pkg trên macOS, hoặc .deb/.rpm trên Linux.

Cách sử dụng

# Đóng gói ứng dụng app.jar thành installer trên Windows
jpackage \
  --input app/ \
  --main-jar app.jar \
  --main-class com.example.Main \
  --name MyApp \
  --type msi \
  --app-version 1.0

# Kết quả: MyApp-1.0.msi có thể cài đặt trên Windows
