Date Time API (java.time)

API phiên bản mới (java.time)

Java 8 giới thiệu gói java.time thay thế cho lớp Date và Calendar cũ vốn bị chỉ trích nhiều. API mới có thiết kế bất biến (immutable), rõ ràng và an toàn với luồng (thread-safe).

Các lớp chính

import java.time.*;
import java.time.format.DateTimeFormatter;

// Chỉ ngày (không có giờ/múi giờ)
LocalDate date = LocalDate.now();           // 2025-03-04
LocalDate birthday = LocalDate.of(1990, 5, 15);

// Chỉ giờ (không có ngày/múi giờ)
LocalTime time = LocalTime.now();           // 16:30:00.123
LocalTime noon = LocalTime.of(12, 0);

// Ngày và giờ (không có múi giờ)
LocalDateTime dateTime = LocalDateTime.now();

// Với múi giờ
ZonedDateTime zoned = ZonedDateTime.now(ZoneId.of("Asia/Ho_Chi_Minh"));

Thao tác với Date/Time

LocalDate today = LocalDate.now();
LocalDate nextWeek = today.plusWeeks(1);
LocalDate lastMonth = today.minusMonths(1);

// Khoảng cách giữa hai ngày
Period period = Period.between(birthday, today);
IO.println("Tuổi: " + period.getYears());

// Định dạng và phân tích
DateTimeFormatter fmt = DateTimeFormatter.ofPattern("dd/MM/yyyy");
String formatted = today.format(fmt); // "04/03/2025"
LocalDate parsed = LocalDate.parse("15/05/1990", fmt);
