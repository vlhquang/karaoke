Biểu thức, Câu lệnh và Khối mã

Biểu thức (Expressions)

Một biểu thức là một cấu trúc được tạo thành từ các biến, toán tử và lời gọi phương thức, được xây dựng theo cú pháp của ngôn ngữ và trả về một giá trị duy nhất.

int cadence = 0;
anArray[0] = 100;
IO.println("Element 1 at index 0: " + anArray[0]);

int result = 1 + 2; // biểu thức là 1 + 2
if (value1 == value2) // biểu thức là value1 == value2

Câu lệnh (Statements)

Câu lệnh tương đương với các câu trong ngôn ngữ tự nhiên. Một câu lệnh tạo thành một đơn vị thực thi hoàn chỉnh. Các loại câu lệnh sau đây có thể được tạo thành bằng cách kết thúc một biểu thức bằng dấu chấm phẩy (;):

  • Biểu thức gán (Assignment expressions)
  • Sử dụng ++ hoặc --
  • Lời gọi phương thức (Method invocations)
  • Biểu thức tạo đối tượng (Object creation expressions)
// câu lệnh gán
aValue = 8933.234;
// câu lệnh tăng giá trị
aValue++;
// câu lệnh gọi phương thức
IO.println("Hello World!");
// câu lệnh tạo đối tượng
Integer x = new Integer(4);

Khối mã (Blocks)

Một khối mã là một nhóm gồm không hoặc nhiều câu lệnh nằm giữa cặp dấu ngoặc nhọn { } và có thể được sử dụng ở bất kỳ đâu mà một câu lệnh đơn lẻ được phép.

class BlockDemo {
    public static void main(String[] args) {
        boolean condition = true;
        if (condition) { // bắt đầu khối 1
            IO.println("Condition is true.");
        } // kết thúc khối 1
        else { // bắt đầu khối 2
            IO.println("Condition is false.");
        } // kết thúc khối 2
    }
}
