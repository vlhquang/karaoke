Stream API

Stream là gì?

Một Stream đại diện cho một chuỗi các phần tử hỗ trợ các thao tác tổng hợp tuần tự và song song. Stream API (giới thiệu trong Java 8) giúp xử lý dữ liệu theo phong cách hàm (functional) một cách ngắn gọn.

Lưu ý: Stream không phải là cấu trúc dữ liệu; nó không lưu trữ phần tử mà chỉ xử lý chúng.

Pipeline của Stream

Một pipeline stream bao gồm 3 phần:

  1. Nguồn (Source): Collection, mảng, hàm tạo (generator).
  2. Thao tác trung gian (Intermediate operations): filter(), map(), sorted()... Trả về một Stream mới. Chúng là "lazy" (chỉ thực thi khi cần).
  3. Thao tác kết thúc (Terminal operations): forEach(), collect(), count(), findFirst()... Kích hoạt thực thi pipeline.

Ví dụ thực tế

List<String> names = List.of("Alice", "Bob", "Charlie", "David", "Anna");

// Lọc, chuyển đổi và thu thập
List<String> result = names.stream()
    .filter(name -> name.startsWith("A")) // ["Alice", "Anna"]
    .map(String::toUpperCase)             // ["ALICE", "ANNA"]
    .sorted()                              // ["ALICE", "ANNA"]
    .collect(Collectors.toList());

// Đếm
long count = names.stream()
    .filter(name -> name.length() > 4)
    .count(); // 3 (Alice, Charlie, David)

// Tổng hợp
int totalLength = names.stream()
    .mapToInt(String::length)
    .sum();
