Tạo và sử dụng đối tượng

Toán tử new

Bạn tạo một đối tượng từ lớp bằng toán tử new:

// Khai báo, tạo và gán tham chiếu trong một dòng
Point originOne = new Point(23, 94);

Câu lệnh này thực hiện 3 thao tác:

  1. Khai báo: Point originOne - khai báo một biến tham chiếu kiểu Point.
  2. Khởi tạo: new - cấp phát bộ nhớ cho đối tượng.
  3. Gán: Trả về tham chiếu tới đối tượng mới và gán cho biến.

Sử dụng đối tượng

Truy cập các trường và phương thức của đối tượng bằng toán tử dấu chấm (.):

// Truy cập trường
IO.println(originOne.x); // 23
IO.println(originOne.y); // 94

// Gọi phương thức
Rectangle rect = new Rectangle(100, 200);
IO.println(rect.getArea()); // 20000

Thu gom rác (Garbage Collection)

Trong Java, bạn không cần phải xóa đối tượng thủ công. Java Runtime Environment (JRE) có một tiến trình nền gọi là Garbage Collector, nó sẽ tự động thu hồi bộ nhớ của các đối tượng không còn được tham chiếu đến.
