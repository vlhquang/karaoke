Bắt đầu với Java

Các thành phần của một ứng dụng Java

Nếu bạn thực sự muốn chạy ngay mã Java đầu tiên của mình, bạn có thể thử Java Playground có sẵn trên trang web này. Bạn có thể thử hầu hết các đoạn mã nhỏ trong đó. Ví dụ: bạn có thể in ra Hello World!, đây chắc chắn là điều đầu tiên bạn muốn thử. Bạn không cần cài đặt bất cứ thứ gì, chỉ cần nhập mã Java, nhấn chạy và xong!

Chu kỳ Biên dịch và Thực thi trong Java

Có một vài bước bạn cần tuân theo để tạo một ứng dụng Java. Hướng dẫn này chỉ cho bạn cách tạo một chương trình Java rất đơn giản. Nếu bạn cần tạo một ứng dụng doanh nghiệp, quy trình sẽ phức tạp hơn nhưng cốt lõi vẫn là những bước đơn giản này.

Bước đầu tiên là viết mã Java trong một trình soạn thảo văn bản.

Sau đó, mã này phải được chuyển đổi sang một định dạng khác mà máy tính của bạn có thể thực thi được. Quá trình chuyển đổi này được thực hiện bởi một phần mềm đặc biệt gọi là trình biên dịch (compiler). Một số ngôn ngữ không có trình biên dịch; Java thì có. Mỗi trình biên dịch là riêng biệt cho từng ngôn ngữ.

Tệp được tạo ra bởi trình biên dịch thường được gọi là tệp nhị phân (binary file) hoặc tệp thực thi (executable file). Trong khi bạn có thể đọc và hiểu mã nguồn, các tệp nhị phân không dành cho con người đọc. Chỉ máy tính mới có thể hiểu được nó.

Tạo chương trình Java đầu tiên

Mã Java bạn viết được lưu trong các tệp văn bản thuần túy. Trong hướng dẫn này, ứng dụng của bạn sẽ được viết trong một tệp văn bản duy nhất. Các ứng dụng lớn hơn có thể yêu cầu hàng nghìn tệp như vậy.

Chúng ta sẽ giả định tên tệp là MyFirstJavaApp.java. Bạn có thể đặt bất kỳ tên nào, miễn là giữ phần mở rộng là .java.

void main() { 
    IO.println("Hello, World!"); 
}

Cài đặt Bộ công cụ phát triển Java - JDK

Để biên dịch và chạy Java, bạn cần Java Development Kit (JDK). JDK chứa nhiều công cụ, bao gồm trình biên dịch và môi trường thực thi.

Bạn có thể tải JDK từ trang: https://jdk.java.net/. Hãy chọn phiên bản mới nhất phù hợp với hệ điều hành của bạn (Windows, Linux, hoặc macOS).

Biên dịch và Chạy chương trình

Sau khi cài đặt JDK, hãy mở terminal hoặc command prompt và di chuyển đến thư mục chứa tệp MyFirstJavaApp.java của bạn.

Kiểm tra xem Java đã được cài đặt chưa bằng lệnh:

> java -version

Để chạy chương trình, hãy nhập lệnh sau:

> java MyFirstJavaApp.java

Nếu chương trình in ra Hello World!, xin chúc mừng! Bạn đã chạy thành công chương trình Java đầu tiên của mình.

Làm cho chương trình có tính tương tác

Hãy nâng cấp chương trình để hỏi tên người dùng:

void main() { 
    IO.println("Chào mừng bạn!"); 
    var name = IO.readln("Tên của bạn là gì? "); 
    IO.println("Xin chào " + name); 
}

Đi xa hơn

Các lập trình viên chuyên nghiệp thường sử dụng Môi trường phát triển tích hợp (IDE) như IntelliJ IDEA, Eclipse hoặc NetBeans để quản lý mã nguồn hiệu quả hơn.
