Sử dụng các tính năng xem trước trong JDK

Tính năng xem trước là gì?

Preview features là các tính năng ngôn ngữ hoặc JVM đã hoàn chỉnh nhưng chưa cố định trong một phiên bản JDK cụ thể. Chúng được đưa vào để cộng đồng phản hồi trước khi ra mắt chính thức.

Cách bật tính năng xem trước

Bạn cần truyền flag --enable-preview khi biên dịch và chạy:

# Biên dịch
javac --enable-preview --release 25 Main.java

# Chạy
java --enable-preview Main

Lưu ý

Tính năng xem trước chỉ nên dùng để thử nghiệm và phản hồi, không nên dùng trong môi trường sản xuất vì chúng có thể thay đổi hoặc bị loại bỏ trong phiên bản tiếp theo.
