Giới thiệu về JavaFX Animations

Transitions

JavaFX cung cấp các lớp Transition sẵn có để tạo animation đơn giản:

// Di chuyển một node từ A đến B
TranslateTransition tt = new TranslateTransition(Duration.millis(1000), myNode);
tt.setFromX(0);
tt.setToX(200);
tt.setCycleCount(Animation.INDEFINITE);
tt.setAutoReverse(true);
tt.play();

// Xoay node
RotateTransition rt = new RotateTransition(Duration.millis(500), myNode);
rt.setByAngle(360);
rt.play();

// Fade in/out
FadeTransition ft = new FadeTransition(Duration.millis(1000), myNode);
ft.setFromValue(1.0);
ft.setToValue(0.0);
ft.play();

Timeline - Animation tùy chỉnh

Timeline timeline = new Timeline(
    new KeyFrame(Duration.ZERO, new KeyValue(node.opacityProperty(), 0)),
    new KeyFrame(Duration.millis(1000), new KeyValue(node.opacityProperty(), 1))
);
timeline.setCycleCount(Animation.INDEFINITE);
timeline.setAutoReverse(true);
timeline.play();
