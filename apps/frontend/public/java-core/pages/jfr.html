JDK Flight Recorder (JFR)

JFR là gì?

JDK Flight Recorder (JFR) là một công cụ profiling và event recording được tích hợp trong JVM từ Java 11+. Nó ghi lại các sự kiện với overhead rất thấp (thường <1%), cho phép chạy trong môi trường sản xuất.

Cách sử dụng

# Bật JFR khi khởi chạy ứng dụng
java -XX:StartFlightRecording=duration=60s,filename=recording.jfr Main

# Bật JFR liên tục (không giới hạn thời gian)
java -XX:StartFlightRecording MyApp

# Dump recording từ tiến trình đang chạy
jcmd <pid> JFR.dump filename=recording.jfr

Phân tích với JDK Mission Control

Tệp .jfr có thể được mở và phân tích bằng JDK Mission Control (jmc) - một công cụ GUI miễn phí từ Oracle và Oracle Labs.
