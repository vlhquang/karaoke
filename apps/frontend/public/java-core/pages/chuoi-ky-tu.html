Chuỗi (Strings)

Tạo chuỗi

Lớp String trong Java được sử dụng để tạo và thao tác chuỗi ký tự. Chuỗi là bất biến (immutable).

// Cách thông thường
String greeting = "Hello world!";

// Dùng constructor
char[] helloArray = {'h', 'e', 'l', 'l', 'o'};
String helloString = new String(helloArray);

Các phương thức thông dụng của String

String s = "Hello World";
IO.println(s.length());        // 11
IO.println(s.charAt(0));       // 'H'
IO.println(s.indexOf("World")); // 6
IO.println(s.toLowerCase());   // "hello world"
IO.println(s.toUpperCase());   // "HELLO WORLD"
IO.println(s.trim());          // Xóa khoảng trắng đầu/cuối
IO.println(s.replace("World", "Java")); // "Hello Java"
IO.println(s.substring(6));    // "World"
IO.println(s.contains("World")); // true

Nối chuỗi

Có thể nối chuỗi bằng toán tử + hoặc phương thức concat().

String first = "Hello";
String second = " World";
String joined = first + second; // "Hello World"
// Hoặc
String joined2 = first.concat(second);

Định dạng chuỗi

Sử dụng String.format() hoặc formatted() để tạo chuỗi có định dạng:

String formatted = String.format("Name: %s, Age: %d", "Java", 30);
// Hoặc (Java 15+)
String formatted2 = "Name: %s, Age: %d".formatted("Java", 30);
