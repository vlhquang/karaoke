Autoboxing và Unboxing

Autoboxing

Autoboxing là quá trình chuyển đổi tự động mà trình biên dịch Java thực hiện giữa kiểu nguyên thủy và lớp bao bọc tương ứng của nó. Ví dụ, chuyển int sang Integer.

// Thay vì phải viết:
Character ch = Character.valueOf('a');
// Trình biên dịch sẽ tự động chuyển:
Character ch = 'a'; // Autoboxing

Unboxing

Unboxing là quá trình ngược lại: chuyển từ lớp bao bọc sang kiểu nguyên thủy. 

Integer i = 42;   // Autoboxing
int j = i;        // Unboxing: Integer -> int
int k = i + 5;    // Unboxing ngầm trước khi cộng

Khi nào xảy ra?

Autoboxing/Unboxing xảy ra tự động khi trình biên dịch phát hiện sự không khớp về kiểu:

  • Khi gán giá trị nguyên thủy vào một biến kiểu bao bọc (và ngược lại).
  • Khi truyền tham số nguyên thủy vào phương thức yêu cầu kiểu bao bọc (và ngược lại).
  • Khi sử dụng các Collection (ví dụ: List<Integer>).
