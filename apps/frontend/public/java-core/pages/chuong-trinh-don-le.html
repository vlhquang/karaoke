Chạy chương trình mã nguồn đơn lẻ

Thực thi chương trình mã nguồn đơn lẻ

Trong JDK 11, Java đã giới thiệu khả năng khởi chạy một chương trình mã nguồn tệp đơn lẻ bằng trình chạy java, và từ JDK 22 đã hỗ trợ đa tệp, mà không cần phải biên dịch mã nguồn một cách rõ ràng trước đó. Điều này hoạt động bằng cách trình chạy java tự động gọi trình biên dịch và lưu trữ mã đã biên dịch trong bộ nhớ.

Đây là một cách tuyệt vời để học cách sử dụng Java hoặc khám phá các tính năng mới trong Java API mà không phải trải qua các bước biên dịch và thực thi mã phức tạp.

Thực thi chương trình đầu tiên của bạn

Để thực thi một chương trình mã nguồn tệp đơn lẻ, lớp đầu tiên được định nghĩa trong tệp nguồn phải chứa phương thức public static void main(String[]) như ví dụ HelloWorld dưới đây:

public class HelloWorld {
    public static void main(String[] args) {
        IO.println("Hello World!");
    }
}

Từ dòng lệnh, HelloWorld có thể được khởi chạy bằng lệnh (giả sử tên tệp là HelloWorld.java):

$ java HelloWorld.java

Nhiều lớp trong cùng một tệp

Nhiều lớp có thể được định nghĩa trong cùng một tệp nguồn nếu cần thiết cho mục đích đóng gói, như trong ví dụ này:

public class MultipleClassesInSameFile {
    public static void main(String[] args) {
        IO.println(GenerateMessage.generateMessage());
        IO.println(AnotherMessage.generateAnotherMessage());
    }
}

class GenerateMessage {
    static String generateMessage() {
        return "Đây là một thông báo";
    }
}

class AnotherMessage {
    static String generateAnotherMessage() {
        return "Đây là một thông báo khác";
    }
}

Khởi chạy chương trình đa tệp

Kể từ khi ra mắt JDK 22, bao gồm JEP 458, trình chạy java hiện có thể khởi chạy một chương trình mã nguồn đa tệp. Điều này cung cấp một sự chuyển đổi dần dần từ các chương trình tệp đơn lẻ đơn giản sang các chương trình phức tạp hơn.

Trình chạy java sẽ tự động tìm kiếm, biên dịch và tải các chương trình đa tệp miễn là vị trí của các tệp nguồn khớp với cấu trúc gói (package) của các lớp được nhập (imported).

Thực thi như một tệp Shebang

Trên hệ điều hành giống Unix, một ứng dụng mã nguồn tệp đơn lẻ cũng có thể được khởi chạy như một tệp shebang giống như một tập lệnh (script). Thêm đường dẫn đến java và phiên bản nguồn vào dòng đầu tiên:

#!/path/to/your/bin/java --source 25
public class HelloJava {
    public static void main(String[] args) {
        IO.println("Xin chào " + args[0]);
    }
}
