Lambda Expressions

Lambda là gì?

Lambda expression (biểu thức lambda) là một hàm ẩn danh (anonymous function) ngắn gọn, có thể được truyền như một tham số. Chúng được giới thiệu trong Java 8 và là nền tảng của lập trình hàm (functional programming) trong Java.

Cú pháp Lambda

// Cú pháp: (tham số) -> biểu_thức
// Hoặc: (tham số) -> { khối_lệnh; }

// Ví dụ 1: Không tham số
Runnable r = () -> IO.println("Hello Lambda!");

// Ví dụ 2: Một tham số
Consumer<String> printer = name -> IO.println("Hello, " + name);

// Ví dụ 3: Nhiều tham số
Comparator<Integer> comp = (a, b) -> a - b;

Functional Interface

Lambda chỉ có thể được sử dụng ở những nơi mong đợi một Functional Interface — là một interface chỉ có đúng một phương thức trừu tượng. Các functional interface phổ biến trong java.util.function:

// Predicate<T>: nhận T, trả về boolean
Predicate<String> isEmpty = s -> s.isEmpty();

// Function<T, R>: nhận T, trả về R
Function<String, Integer> length = s -> s.length();

// Consumer<T>: nhận T, không trả về gì
Consumer<String> print = s -> IO.println(s);

// Supplier<T>: không nhận tham số, trả về T
Supplier<String> greeting = () -> "Hello!";

Ví dụ thực tế

List<String> names = List.of("Alice", "Bob", "Charlie");

// Sắp xếp bằng lambda
names.stream()
     .sorted((a, b) -> a.compareTo(b))
     .forEach(name -> IO.println(name));
