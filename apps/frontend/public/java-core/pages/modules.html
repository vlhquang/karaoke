Modules trong Java (JPMS)

Java Platform Module System (JPMS)

Được giới thiệu trong Java 9 (Project Jigsaw), Hệ thống Module Java (JPMS) cho phép chia nhỏ JDK và ứng dụng thành các module riêng biệt, cải thiện đóng gói và bảo mật.

Khai báo module (module-info.java)

Mỗi module được định nghĩa bởi tệp module-info.java tại thư mục gốc của module:

// module-info.java
module com.example.myapp {
    // Phụ thuộc vào module khác
    requires java.sql;
    requires transitive java.logging;

    // Xuất package để các module khác sử dụng
    exports com.example.myapp.api;
    exports com.example.myapp.model;

    // Mở package cho reflection (ví dụ: cho framework)
    opens com.example.myapp.config to spring.core;
}

Lợi ích của Module System

  • Strong encapsulation: Mã nội bộ không thể bị truy cập từ bên ngoài module.
  • Reliable dependencies: Phụ thuộc được khai báo rõ ràng, không còn "classpath hell".
  • Smaller deployments: Tạo runtime nhỏ gọn với chỉ các module cần thiết (jlink).
  • Improved security: Giảm bề mặt tấn công.
