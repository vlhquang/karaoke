Lớp lồng nhau (Nested Classes)

Giới thiệu

Ngôn ngữ Java cho phép bạn định nghĩa một lớp bên trong một lớp khác. Lớp đó được gọi là lớp lồng nhau (nested class).

class OuterClass {
    // ...
    class NestedClass {
        // ...
    }
}

Các loại lớp lồng nhau

Lớp lồng nhau được chia thành hai loại:

  • Lớp lồng tĩnh (Static Nested Class): Được khai báo với từ khóa static. Nó không thể truy cập trực tiếp vào các thành viên instance của lớp bên ngoài.
  • Lớp nội bộ (Inner Class): Không có từ khóa static. Nó có thể truy cập mọi trường và phương thức của lớp bên ngoài, kể cả các thành viên private.

Khi nào nên dùng?

  • Khi chỉ có một lớp sử dụng lớp đó (nhóm logic liên quan).
  • Để tăng cường đóng gói (encapsulation).
  • Để tạo mã ngắn gọn hơn (ví dụ với Anonymous Classes và Lambdas).
