Java I/O API

Tổng quan về I/O

Java cung cấp hai bộ API chính để làm việc với tệp tin và luồng dữ liệu:

  • java.io: API I/O truyền thống với các lớp như File, InputStream, OutputStream.
  • java.nio.file (NIO.2): API hiện đại hơn, linh hoạt hơn với Path, Files.

Làm việc với đường dẫn (Path)

import java.nio.file.Path;
import java.nio.file.Paths;

// Tạo đường dẫn
Path path = Path.of("C:/Users/user/documents/file.txt");

IO.println(path.getFileName()); // file.txt
IO.println(path.getParent());   // C:/Users/user/documents
IO.println(path.getRoot());     // C:/

Lớp Files - Tác vụ phổ biến

import java.nio.file.Files;
import java.nio.file.Path;

Path path = Path.of("data.txt");

// Đọc toàn bộ file thành String
String content = Files.readString(path);

// Đọc tất cả dòng
List<String> lines = Files.readAllLines(path);

// Ghi file
Files.writeString(path, "Hello World");

// Kiểm tra
boolean exists = Files.exists(path);
boolean isDir = Files.isDirectory(path);

// Sao chép và xóa
Files.copy(path, Path.of("backup.txt"));
Files.delete(path);
