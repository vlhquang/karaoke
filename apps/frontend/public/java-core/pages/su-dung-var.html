Sử dụng định danh kiểu var

Từ khóa var

Bắt đầu từ Java 10, bạn có thể sử dụng định danh var thay cho kiểu dữ liệu rõ ràng khi khai báo biến cục bộ. Điều này được gọi là Local Variable Type Inference (Suy luận kiểu biến cục bộ).

Khi sử dụng var, trình biên dịch Java sẽ tự động suy luận kiểu của biến dựa trên giá trị khởi tạo của nó.

Ví dụ sử dụng var

// Thay vì viết:
String message = "Hello World";
// Bạn có thể viết:
var message = "Hello World"; // Trình biên dịch suy luận kiểu String

// Ví dụ với danh sách
var list = new ArrayList<String>(); // Suy luận kiểu ArrayList<String>

Các hạn chế khi dùng var

Mặc dù hữu ích, var có một số quy tắc và hạn chế quan trọng:

  • Chỉ dùng cho biến cục bộ: Bạn không thể dùng var cho các trường (fields) của lớp, tham số phương thức, hoặc kiểu trả về của phương thức.
  • Phải khởi tạo ngay: Biến var phải được khởi tạo ngay khi khai báo để trình biên dịch có cơ sở suy luận kiểu.
  • Không thể khởi tạo null: var x = null; là không hợp lệ vì không thể suy luận kiểu từ null.
  • Không thể dùng cho mảng khởi tạo kiểu {}: var a = {1, 2, 3}; là không hợp lệ.
