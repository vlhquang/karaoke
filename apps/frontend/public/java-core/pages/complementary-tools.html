Các công cụ JDK bổ sung

jextract

jextract là công cụ hỗ trợ FFM API, tự động tạo các binding Java từ header file C (.h). Điều này giúp đơn giản hóa việc tích hợp với thư viện native.

# Tạo Java binding từ header file C
jextract --target-package com.example.native \
         -l mylib \
         mylib.h

Các công cụ khác

  • jshell: REPL (Read-Eval-Print Loop) tương tác.
  • jdeprscan: Quét sử dụng API đã bị deprecated.
  • jcmd: Gửi lệnh chẩn đoán đến tiến trình JVM đang chạy.
