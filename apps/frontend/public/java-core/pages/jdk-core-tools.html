Các công cụ JDK cốt lõi

Bộ công cụ cốt lõi

JDK cung cấp nhiều công cụ dòng lệnh quan trọng:

  • javac: Trình biên dịch Java, chuyển tệp .java thành bytecode .class.
  • java: Java launcher, khởi chạy ứng dụng Java.
  • jar: Quản lý tệp JAR (Java ARchive).
  • javadoc: Tạo tài liệu API HTML từ comment trong code.
  • jdeps: Phân tích phụ thuộc module và package.

javac - Biên dịch

# Biên dịch một tệp
javac HelloWorld.java

# Biên dịch với classpath
javac -cp lib/*.jar src/Main.java -d out/

# Biên dịch cho phiên bản Java cụ thể
javac --release 21 Main.java
