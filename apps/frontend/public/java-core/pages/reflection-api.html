The Reflection API

Đi sâu vào Reflection API

Reflection API trong java.lang.reflect cung cấp các lớp như Class, Field, Method, Constructor cho phép kiểm tra và sửa đổi hành vi runtime.

Truy cập thành viên private

class Secret {
    private String hidden = "Bí mật";
}

Secret obj = new Secret();
Field field = Secret.class.getDeclaredField("hidden");
field.setAccessible(true); // Bypass access control
String value = (String) field.get(obj);
IO.println(value); // "Bí mật"

Lưu ý: Kể từ Java 9, truy cập reflection vào các class trong module bị hạn chế hơn. Bạn cần module export/open để cho phép reflection.
