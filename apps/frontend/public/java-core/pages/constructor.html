Constructor

Constructor là gì?

Một lớp chứa các constructor được gọi khi tạo mới một đối tượng. Constructor có cùng tên với lớp và không có kiểu trả về.

public class Bicycle {
    private int gear;
    private int speed;

    // Constructor
    public Bicycle(int startGear, int startSpeed) {
        gear = startGear;
        speed = startSpeed;
    }
}

Sử dụng Constructor

Khi tạo đối tượng bằng toán tử new, bạn truyền tham số cho constructor:

Bicycle myBike = new Bicycle(5, 25);

Constructor không tham số

Nếu bạn không định nghĩa bất kỳ constructor nào, Java sẽ tự động cung cấp một constructor không có tham số (no-argument constructor), khởi tạo tất cả các trường về giá trị mặc định. Tuy nhiên, nếu bạn đã khai báo ít nhất một constructor, constructor mặc định này sẽ không được cung cấp.
