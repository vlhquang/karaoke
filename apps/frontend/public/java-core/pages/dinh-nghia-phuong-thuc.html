Định nghĩa phương thức (Defining Methods)

Cấu trúc của một phương thức

Một phương thức (method) trong Java có các thành phần:

// [phạm vi truy cập] [kiểu trả về] [tên phương thức]([tham số])
public double calculateAnswer(double wingSpan, int numberOfEngines) {
    // thân phương thức
    return wingSpan * numberOfEngines;
}

Quy ước đặt tên phương thức

  • Bắt đầu bằng chữ thường, dùng camelCase.
  • Để dễ đọc, nên dùng động từ ở đầu: run(), getColor(), setName(name).

Nạp chồng phương thức (Overloading)

Java cho phép bạn có nhiều phương thức trùng tên trong cùng một lớp, miễn là chúng có tham số khác nhau. Trình biên dịch sẽ phân biệt chúng dựa trên chữ ký (signature).

public class DataProcessor {
    public void process(String s) { /* ... */ }
    public void process(int i) { /* ... */ }
    public void process(String s, int i) { /* ... */ }
}
