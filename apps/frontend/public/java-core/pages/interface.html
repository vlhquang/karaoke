Interface trong Java

Interface là gì?

Một interface là một "hợp đồng" định nghĩa những gì một lớp có thể làm, mà không cần lo đến cách nó làm. Một interface chứa các khai báo phương thức trừu tượng (abstract), các phương thức mặc định (default), phương thức tĩnh (static), và các hằng số (constants).

public interface Flyable {
    void fly(); // phương thức trừu tượng
}

Triển khai Interface (implements)

Một lớp triển khai interface bằng từ khóa implements và bắt buộc phải cài đặt tất cả các phương thức trừu tượng của interface đó.

public class Bird implements Flyable {
    @Override
    public void fly() {
        System.out.println("Bird is flying");
    }
}

Interface vs Abstract Class

  • Một lớp chỉ có thể kế thừa (extends) một lớp cha.
  • Nhưng một lớp có thể triển khai (implements) nhiều interface cùng lúc.

Dùng Interface như một kiểu dữ liệu

Khi một lớp triển khai một interface, bạn có thể sử dụng tên interface như một kiểu dữ liệu để khai báo biến tham chiếu đến đối tượng của lớp đó. Đây là nền tảng của tính đa hình (polymorphism).

Flyable myBird = new Bird();
myBird.fly();
