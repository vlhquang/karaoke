Xử lý ngoại lệ (Exceptions)

Ngoại lệ là gì?

Một ngoại lệ (exception) là một sự kiện xảy ra trong quá trình thực thi chương trình, làm gián đoạn luồng lệnh bình thường. Khi xảy ra lỗi, Java tạo ra một đối tượng ngoại lệ và "ném" (throw) nó.

Khối try-catch-finally

try {
    // Đặt code có thể gây lỗi ở đây
    int[] arr = {1, 2, 3};
    IO.println(arr[10]); // Lỗi: ArrayIndexOutOfBoundsException
} catch (ArrayIndexOutOfBoundsException e) {
    // Xử lý lỗi
    IO.println("Lỗi: Chỉ số mảng vượt quá giới hạn: " + e.getMessage());
} finally {
    // Code này luôn được thực thi dù có lỗi hay không
    IO.println("Khối finally luôn chạy.");
}

Phân loại ngoại lệ

  • Checked Exceptions: Phải được xử lý hoặc khai báo (ví dụ: IOException). Trình biên dịch kiểm tra.
  • Unchecked Exceptions (RuntimeException): Không bắt buộc xử lý (ví dụ: NullPointerException, ArrayIndexOutOfBoundsException).
  • Errors: Vấn đề nghiêm trọng của JVM (ví dụ: OutOfMemoryError), thường không thể phục hồi.

Ném ngoại lệ (throw & throws)

// Khai báo phương thức có thể ném ngoại lệ
public void checkAge(int age) throws IllegalArgumentException {
    if (age < 0) {
        throw new IllegalArgumentException("Tuổi không hợp lệ: " + age);
    }
    IO.println("Tuổi hợp lệ: " + age);
}
