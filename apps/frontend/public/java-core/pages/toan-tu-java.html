Sử dụng các Toán tử trong Java

Toán tử là gì?

Toán tử là các ký hiệu đặc biệt thực hiện các thao tác cụ thể trên một, hai hoặc ba toán hạng, sau đó trả về một kết quả.

Toán tử gán (Assignment Operator)

Toán tử gán đơn giản nhất là =, dùng để gán giá trị bên phải cho toán hạng bên trái. 

int speed = 0;
int gear = 1;

Toán tử số học (Arithmetic Operators)

Java cung cấp các toán tử để thực hiện cộng, trừ, nhân, chia và chia lấy dư:

  • + : Cộng (cũng dùng để nối chuỗi)
  • - : Trừ
  • * : Nhân
  • / : Chia
  • % : Chia lấy dư
int result = 1 + 2; // result là 3
result = result - 1; // result là 2
result = result % 7; // result là 3 (số dư của 10 / 7)

Toán tử đơn phân (Unary Operators)

Các toán tử này chỉ cần một toán hạng duy nhất:

  • ++ : Tăng giá trị lên 1
  • -- : Giảm giá trị đi 1
  • ! : Đảo ngược giá trị logic (boolean)
  • - : Đảo dấu giá trị số
int i = 3;
i++; // i là 4
++i; // i là 5
IO.println(!true); // In ra false

Toán tử so sánh (Relational Operators)

Dùng để xác định mối quan hệ giữa hai toán hạng:

  • == : Bằng nhau
  • != : Khác nhau
  • > : Lớn hơn
  • >= : Lớn hơn hoặc bằng
  • < : Nhỏ hơn
  • <= : Nhỏ hơn hoặc bằng

Toán tử điều kiện (Conditional Operators)

Các toán tử logic && (VÀ) và || (HOẶC) được sử dụng trên hai biểu thức boolean. Chúng có cơ chế "ngắt sớm" (short-circuiting).

Toán tử tam phân ?: là cách viết tắt của câu lệnh if-then-else:

result = someCondition ? value1 : value2;
