JWebserver - Java Simple Web Server

JWebserver là gì?

Được giới thiệu trong Java 18, jwebserver là một web server HTTP đơn giản tích hợp trong JDK, được thiết kế cho mục đích prototype, testing và debugging. Nó phục vụ các tệp tĩnh (static files) từ thư mục hiện tại.

Cách sử dụng

# Chạy web server trên cổng 8000 (mặc định)
jwebserver

# Chỉ định cổng và thư mục
jwebserver -p 9090 -d /path/to/directory

# Truy cập: http://localhost:8000

Server sẽ phục vụ tất cả các tệp trong thư mục được chỉ định và ghi lại mỗi request vào console.
