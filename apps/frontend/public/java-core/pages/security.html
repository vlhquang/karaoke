Bảo mật với JDK Libraries

Tổng quan

JDK cung cấp các thư viện bảo mật toàn diện trong gói java.security, hỗ trợ mã hóa, chứng thực số, quản lý khóa và nhiều hơn nữa.

Băm (Hashing)

import java.security.MessageDigest;

MessageDigest md = MessageDigest.getInstance("SHA-256");
byte[] hash = md.digest("Hello World".getBytes());

// Chuyển thành hex string
StringBuilder hex = new StringBuilder();
for (byte b : hash) {
    hex.append(String.format("%02x", b));
}
IO.println(hex.toString());

HTTPS với HttpClient

Java HttpClient hỗ trợ HTTPS tích hợp sẵn. Khi bạn gửi request đến một URL https://, JVM tự động xử lý SSL/TLS handshake và xác thực chứng chỉ.
