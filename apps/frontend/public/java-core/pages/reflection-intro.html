Giới thiệu Java Reflection

Reflection là gì?

Reflection cho phép một đối tượng "nhìn vào gương" và khám phá các trường, phương thức và constructor của mình tại thời điểm runtime. Bạn có thể đọc/ghi trường, gọi phương thức và tạo đối tượng mới bằng cách gọi constructor.

Ví dụ cơ bản

// Lấy thông tin class tại runtime
Class<?> clazz = String.class;
IO.println(clazz.getName()); // "java.lang.String"

// Liệt kê tất cả methods
Method[] methods = clazz.getMethods();
for (Method m : methods) {
    IO.println(m.getName());
}

// Gọi phương thức động
Method toUpperCase = clazz.getMethod("toUpperCase");
Object result = toUpperCase.invoke("hello world");
IO.println(result); // "HELLO WORLD"
