The Foreign Function and Memory API (FFM)

FFM API là gì?

Foreign Function and Memory (FFM) API (ra mắt chính thức trong Java 22) cho phép chương trình Java tương tác với code và dữ liệu bên ngoài Java runtime mà không cần dùng JNI (Java Native Interface) vốn phức tạp và tiềm ẩn rủi ro.

Trường hợp sử dụng

  • Gọi hàm trong thư viện native C/C++ (.dll, .so)
  • Làm việc với bộ nhớ ngoài heap (off-heap memory)
  • Tích hợp với code native hiệu suất cao
