JShell - Công cụ Shell của Java

Giới thiệu về jshell

jshell là một công cụ tương tác cho phép đánh giá các khai báo, câu lệnh và biểu thức của ngôn ngữ lập trình Java trong một vòng lặp read-eval-print (REPL).

Nó cung cấp một cách để tìm hiểu ngôn ngữ, khám phá các mã và API lạ, và tạo mẫu mã phức tạp một cách nhanh chóng. Các câu lệnh Java, định nghĩa biến, định nghĩa phương thức, định nghĩa lớp, câu lệnh import và biểu thức đều được chấp nhận. Các đoạn mã được nhập vào được gọi là snippets.

Khởi động và Kết thúc phiên jshell

jshell đi kèm với JDK. Để bắt đầu một phiên làm việc, hãy nhập jshell trên dòng lệnh. Một thông báo chào mừng sẽ hiện ra, cùng với dấu nhắc để nhập các lệnh và đoạn mã.

$ jshell
|  Welcome to JShell -- Version 25
|  For an introduction type: /help intro

jshell>

Để kết thúc phiên làm việc, hãy sử dụng lệnh /exit.

jshell> /exit
|  Goodbye

Nhập các đoạn mã (Snippets)

Ví dụ sau đây cho thấy hai biến và một phương thức được định nghĩa, sau đó phương thức được chạy. Lưu ý rằng một biến tạm (scratch variable) sẽ tự động được tạo để giữ kết quả nếu không có biến nào được cung cấp.

jshell> int a=4
a ==> 4

jshell> int b=8
b ==> 8

jshell> int square(int i1) {
   ...> return i1 * i1;
   ...> }
|  created method square(int)

jshell> square(b)
$5 ==> 64

Thay đổi các đoạn mã

Bạn có thể thay đổi định nghĩa của một biến, phương thức hoặc lớp bằng cách nhập lại nó. Bạn có thể sử dụng phím mũi tên lên để lấy lại các mục đã nhập trước đó để tránh phải nhập lại toàn bộ. 

jshell> String grade(int testScore) {
   ...> if (testScore >= 90) {
   ...> return "Pass";
   ...> }
   ...> return "Fail";
   ...> }
|  created method grade(int)

jshell> grade(88)
$3 ==> "Fail"

Để thay đổi phương thức grade cho phép nhiều sinh viên vượt qua hơn, hãy nhập lại định nghĩa phương thức và thay đổi điểm đạt thành 80:

jshell> String grade(int testScore) { ... }
|  modified method grade(int)

jshell> grade(88)
$5 ==> "Pass"

Các lệnh hữu ích trong jshell

  • /list: Chỉnh sửa hoặc liệt kê các đoạn mã đã nhập.
  • /edit: Mở một trình soạn thảo để chỉnh sửa các đoạn mã.
  • /drop: Xóa một đoạn mã.
  • /history: Hiển thị những gì đã được nhập trong phiên này.
  • /open: Mở một tệp hoặc tập lệnh (script).
