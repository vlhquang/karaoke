Số và Chuỗi

Phần này bao gồm các kiểu số với kiểu nguyên thủy và lớp bao bọc (wrapper), cách định dạng số, sử dụng hàm toán học, và xử lý chuỗi ký tự.

Số (Numbers)

Sử dụng số với kiểu nguyên thủy và wrapper, định dạng số, hàm toán học.

Ký tự (Characters)

Sử dụng ký tự, hiểu giá trị char và code point.

Chuỗi (Strings)

Tạo chuỗi ký tự, khám phá lớp String để thao tác chuỗi.

String Builders

Sử dụng StringBuilder để tạo chuỗi ký tự.

Autoboxing và Unboxing

Chuyển đổi tự động giữa kiểu nguyên thủy và lớp bao bọc tương ứng.
