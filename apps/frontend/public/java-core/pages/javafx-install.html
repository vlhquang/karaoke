Tải và cài đặt JavaFX

JavaFX là gì?

JavaFX là framework để xây dựng ứng dụng desktop phong phú (rich client) trong Java. Kể từ Java 11, JavaFX không còn là một phần của JDK mà được phân phối riêng qua openjfx.io.

Tải JavaFX SDK

  1. Truy cập https://openjfx.io/.
  2. Chọn phiên bản JavaFX phù hợp với JDK của bạn.
  3. Chọn hệ điều hành và type SDK.
  4. Tải và giải nén vào một thư mục (ví dụ: C:\javafx-sdk).

Chạy ứng dụng JavaFX

# Biên dịch
javac --module-path /path/to/javafx-sdk/lib \
      --add-modules javafx.controls,javafx.fxml \
      Main.java

# Chạy
java --module-path /path/to/javafx-sdk/lib \
     --add-modules javafx.controls,javafx.fxml \
     Main
