Rẽ nhánh với câu lệnh Switch

Câu lệnh Switch là gì?

Câu lệnh switch cho phép thực hiện nhiều đường dẫn thực thi khác nhau. Nó nhận một biến lựa chọn (selector variable) làm đối số và sử dụng giá trị của biến này để chọn đường dẫn sẽ được thực thi.

Các kiểu dữ liệu được hỗ trợ cho biến lựa chọn:

  • Các kiểu nguyên thủy: byte, short, char, và int.
  • Các kiểu bao bọc (Wrapper types): Character, Byte, Short, và Integer.
  • Các kiểu liệt kê (Enumerated types - Enum).
  • Kiểu String.

Lưu ý: boolean, long, float, và double không được hỗ trợ làm biến lựa chọn.

Ví dụ về Switch

int quarter = 1;
String quarterLabel;
switch (quarter) {
    case 0:
        quarterLabel = "Q1 - Mùa Đông";
        break;
    case 1:
        quarterLabel = "Q2 - Mùa Xuân";
        break;
    case 2:
        quarterLabel = "Q3 - Mùa Hạ";
        break;
    case 3:
        quarterLabel = "Q4 - Mùa Thu";
        break;
    default:
        quarterLabel = "Quý không hợp lệ";
        break;
}

Từ khóa break và Cơ chế Fall-through

Mỗi câu lệnh break sẽ kết thúc câu lệnh switch bao quanh nó. Luồng điều khiển sẽ tiếp tục với câu lệnh đầu tiên sau khối switch.

Nếu thiếu break, các câu lệnh trong khối switch sẽ bị "trôi" (fall-through). Tất cả các câu lệnh sau nhãn case khớp sẽ được thực thi theo trình tự cho đến khi gặp break.

Sử dụng String trong Switch

Trong Java SE 7 và mới hơn, bạn có thể sử dụng đối tượng String trong biểu thức switch.

String month = "August";
switch (month.toLowerCase()) {
    case "january": monthNumber = 1; break;
    case "february": monthNumber = 2; break;
    // ...
    default: monthNumber = 0; break;
}
