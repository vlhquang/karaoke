Số (Numbers) trong Java

Lớp bao bọc (Wrapper Classes)

Mỗi kiểu số nguyên thủy có một lớp bao bọc tương ứng trong gói java.lang:

// Kiểu nguyên thủy  → Lớp bao bọc
// byte              → Byte
// short             → Short
// int               → Integer
// long              → Long
// float             → Float
// double            → Double

Integer x = 5;  // Autoboxing từ int sang Integer
int y = x;      // Unboxing từ Integer sang int

Các phương thức của lớp bao bọc

Các lớp bao bọc cung cấp nhiều phương thức hữu ích:

// Chuyển đổi
int i = Integer.parseInt("42");
String s = String.valueOf(123);
double d = Double.parseDouble("3.14");

// Giới hạn kiểu
IO.println(Integer.MAX_VALUE); // 2147483647
IO.println(Integer.MIN_VALUE); // -2147483648

Lớp Math

Lớp java.lang.Math cung cấp các hàm toán học tĩnh:

Math.abs(-5);         // 5
Math.max(10, 20);     // 20
Math.min(10, 20);     // 10
Math.pow(2, 10);      // 1024.0
Math.sqrt(16);        // 4.0
Math.floor(3.7);      // 3.0
Math.ceil(3.2);       // 4.0
Math.round(3.5);      // 4
Math.random();        // Số ngẫu nhiên từ [0, 1)
