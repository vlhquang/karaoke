Tạo Runtime và Application Images với JLink

JLink là gì?

jlink là công cụ dòng lệnh cho phép bạn tạo một custom runtime image chỉ chứa các module JDK mà ứng dụng của bạn thực sự cần. Kết quả là một runtime nhỏ gọn có thể phân phối cùng ứng dụng.

Cách dùng jlink

# Tạo runtime image chỉ với các module cần thiết
jlink \
  --module-path $JAVA_HOME/jmods \
  --add-modules java.base,java.sql \
  --output my-runtime

# Chạy ứng dụng với runtime tùy chỉnh
my-runtime/bin/java -m com.example.myapp/com.example.Main
