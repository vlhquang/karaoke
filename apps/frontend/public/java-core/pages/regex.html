Biểu thức chính quy (Regular Expressions)

Cú pháp cơ bản

Biểu thức chính quy (regex) là một chuỗi ký tự đặc biệt dùng để tìm kiếm và thao tác văn bản theo mẫu.

.      // Khớp với bất kỳ ký tự nào
\d     // Khớp chữ số [0-9]
\w     // Khớp ký tự từ [a-zA-Z0-9_]
\s     // Khớp khoảng trắng
^      // Bắt đầu chuỗi
$      // Kết thúc chuỗi
*      // 0 hoặc nhiều lần
+      // 1 hoặc nhiều lần
?      // 0 hoặc 1 lần

Sử dụng trong Java

import java.util.regex.*;

String text = "Số điện thoại: 0912345678";
Pattern pattern = Pattern.compile("0\\d{9}");
Matcher matcher = pattern.matcher(text);

if (matcher.find()) {
    IO.println("Tìm thấy: " + matcher.group()); // "0912345678"
}

// Validate email đơn giản
boolean isEmail = "user@example.com".matches("[\\w.]+@[\\w.]+\\.\\w+");
IO.println(isEmail); // true
