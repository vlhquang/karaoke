Công cụ giám sát (Monitoring Tools)

JConsole

JConsole là công cụ giám sát đồ họa tích hợp trong JDK, giúp bạn giám sát hiệu suất và tài nguyên của ứng dụng Java đang chạy thông qua giao diện đồ họa (GUI).

# Khởi chạy JConsole
jconsole

# Hoặc kết nối đến một process cụ thể
jconsole <pid>

jstat - Thống kê JVM

# Hiển thị thống kê Garbage Collection mỗi 1 giây
jstat -gc <pid> 1000

# Hiển thị tải class
jstat -class <pid>

jmap và jstack

  • jmap: In thống kê heap và dump bộ nhớ của tiến trình Java.
  • jstack: In trạng thái stack trace của tất cả luồng trong tiến trình Java.
