Sự tiến hóa của nền tảng Java

Java đã đi bao xa?

Java đã phát triển rất nhiều kể từ phiên bản 8. Oracle chuyển sang chu kỳ phát hành 6 tháng một lần kể từ Java 9, mang đến các tính năng mới liên tục thay vì chờ đợi hàng năm.

Các mốc quan trọng

  • Java 8 (2014): Lambda, Stream API, Date/Time API, Optional
  • Java 9 (2017): Module System (JPMS), JShell
  • Java 11 (2018, LTS): HTTP Client, String methods mới
  • Java 14 (2020): Records (preview), Switch Expressions
  • Java 16 (2021): Records chính thức, Pattern Matching for instanceof
  • Java 17 (2021, LTS): Sealed Classes, Strong Encapsulation
  • Java 21 (2023, LTS): Virtual Threads, Pattern Matching for switch, Sequenced Collections
  • Java 25 (2025, LTS): Phiên bản LTS mới nhất

Phiên bản LTS là gì?

Long-Term Support (LTS) là các phiên bản được hỗ trợ trong thời gian dài (nhiều năm), phù hợp cho môi trường sản xuất. Các phiên bản LTS khuyến nghị hiện nay là Java 17, Java 21, và Java 25.
