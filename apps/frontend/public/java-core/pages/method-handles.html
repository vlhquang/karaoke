Giới thiệu Method Handles

Method Handle là gì?

Method Handle là một tham chiếu trực tiếp, được kiểm tra kiểu, đến một phương thức, hàm khởi tạo, truy cập trường, hoặc thao tác chuyển đổi kiểu tương tự. Chúng là phương án hiệu suất cao thay thế cho Reflection API.

Ví dụ

import java.lang.invoke.*;

MethodHandles.Lookup lookup = MethodHandles.lookup();

// Lấy handle cho phương thức String.toUpperCase()
MethodType mt = MethodType.methodType(String.class);
MethodHandle toUpperCase = lookup.findVirtual(String.class, "toUpperCase", mt);

// Gọi method handle
String result = (String) toUpperCase.invoke("hello");
IO.println(result); // "HELLO"
