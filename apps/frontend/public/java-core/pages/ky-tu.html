Ký tự (Characters)

Kiểu char

Kiểu nguyên thủy char là ký tự Unicode 16-bit. Giá trị của nó nằm trong khoảng từ '\u0000' (0) đến '\uffff' (65535).

char capital_C = 'C';
char copyrightSymbol = '\u00A9'; // ký tự ©

Lớp Character

Lớp Character (bao bọc của char) cung cấp nhiều phương thức tĩnh hữu ích để kiểm tra và xử lý ký tự:

Character.isLetter('A');    // true
Character.isDigit('3');    // true
Character.isWhitespace(' '); // true
Character.isUpperCase('A'); // true
Character.isLowerCase('a'); // true
Character.toUpperCase('a'); // 'A'
Character.toLowerCase('A'); // 'a'
