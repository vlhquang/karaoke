Enum (Kiểu liệt kê)

Enum là gì?

Một enum type là một kiểu dữ liệu đặc biệt cho phép một biến chỉ được là một trong số các hằng số được định nghĩa trước. Enum phù hợp để biểu diễn các tập hợp cố định như ngày trong tuần, mùa, hướng la bàn, vv.

public enum Day {
    SUNDAY, MONDAY, TUESDAY, WEDNESDAY,
    THURSDAY, FRIDAY, SATURDAY
}

Sử dụng Enum

Day today = Day.MONDAY;

switch (today) {
    case MONDAY:
        IO.println("Đầu tuần làm việc.");
        break;
    case FRIDAY:
        IO.println("Gần cuối tuần rồi!");
        break;
    case SATURDAY:
    case SUNDAY:
        IO.println("Cuối tuần!");
        break;
    default:
        IO.println("Giữa tuần.");
        break;
}

Enum có phương thức và trường

Trong Java, enum có thể có constructor, trường và phương thức, giúp bạn thêm chức năng vào mỗi hằng số.

public enum Planet {
    MERCURY(3.303e+23, 2.4397e6),
    VENUS(4.869e+24, 6.0518e6),
    EARTH(5.976e+24, 6.37814e6);

    private final double mass;
    private final double radius;

    Planet(double mass, double radius) {
        this.mass = mass;
        this.radius = radius;
    }

    double surfaceGravity() {
        final double G = 6.67300E-11;
        return G * mass / (radius * radius);
    }
}
