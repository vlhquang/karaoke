String Builders

Tại sao cần StringBuilder?

Đối tượng String là bất biến (immutable). Mỗi khi bạn sửa đổi một String (nối chuỗi, thay thế...), Java thực ra tạo ra một đối tượng String mới. Điều này gây lãng phí bộ nhớ và hiệu suất khi phải nối chuỗi nhiều lần trong vòng lặp.

StringBuilder giải quyết vấn đề này bằng cách cung cấp một chuỗi có thể thay đổi được (mutable).

Sử dụng StringBuilder

StringBuilder sb = new StringBuilder();
sb.append("Hello");
sb.append(" ");
sb.append("World");
IO.println(sb.toString()); // "Hello World"

// Các phương thức khác
sb.insert(5, ",");         // Chèn vào vị trí 5
sb.delete(5, 6);           // Xóa ký tự
sb.reverse();              // Đảo ngược chuỗi
sb.replace(0, 5, "Hi");   // Thay thế đoạn
