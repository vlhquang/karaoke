Tạo lớp (Creating Classes)

Cấu trúc của một lớp

Đây là cú pháp tổng quát để định nghĩa một lớp trong Java:

class MyClass {
    // Các trường (fields/variables)
    int x = 5;
    
    // Phương thức (methods)
    void print() {
        System.out.println("x = " + x);
    }
}

Khai báo trường thành viên (Member Variables)

Các biến bên trong một lớp gọi là trường (fields). Mỗi trường có:

  • Phạm vi truy cập: public, private, protected
  • Kiểu dữ liệu: int, String, ...
  • Tên trường
public class Bicycle {
    // Trường static (biến lớp)
    public static int numberOfBicycles = 0;
    
    // Trường instance (biến đối tượng)
    public int cadence;
    public int speed;
    private int gear; // phạm vi private
}

Kiểm soát truy cập

  • public: Có thể truy cập từ bất kỳ đâu.
  • private: Chỉ có thể truy cập trong cùng lớp.
  • protected: Truy cập được trong cùng package và lớp con.
  • (Không khai báo - package-private): Truy cập được trong cùng package.
