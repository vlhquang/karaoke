Annotations (Chú thích)

Annotations là gì?

Annotations là một dạng metadata cung cấp thông tin về chương trình mà không ảnh hưởng trực tiếp đến hoạt động của code. Chúng bắt đầu bằng ký tự @.

Các Annotation phổ biến

@Override      // Ghi đè phương thức từ lớp cha
@Deprecated    // Đánh dấu code lỗi thời
@SuppressWarnings("unchecked") // Tắt cảnh báo

// Annotation tự định nghĩa
@interface MyAnnotation {
    String value() default "default";
    int count() default 1;
}

// Sử dụng
@MyAnnotation(value = "test", count = 3)
public class MyClass { }
